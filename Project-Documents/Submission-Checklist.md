# Submission Checklist

- `requirements.txt` is present for the Python scripts.
- `Project-Documents/R-Environment.md` records the verified R versions and packages.
- README documents setup, data placement, training, ensemble rebuilding, and validation.
- `artifacts/submission_final` is the official local final artifact bundle.
- `comparison_report.json` and `submission_summary.json` are present in the final artifact bundle.
- Final report is stored in `Project-Documents/Final-Project-Report.md`.
- Raw data stays local under `Data/` and is not committed.
- Before handing in, commit the final repo state and include the final artifact metrics in the written deliverable.
